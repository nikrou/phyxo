Phxyo est un logiciel vous permettant de réaliser simplement une galerie photos.
Phyxo vous permet d'héberger vos photos sur votre serveur.

Fonctionnalités

  • Phyxo peut-être enrichi avec des extensions, des thèmes,...
  • Vous pouvez ajouter des tags sur vos photos mais aussi autoriser vos visiteurs à tagger vos photos
  • Une gestion fine des permissions vous permet de définir qui peut voir une photo donnée
  • Phyxo peut récupérer les informations EXIF de vos photos (date, auteur,...)
  • Vos visiteurs peuvent noter, commenter vos photos
  • Une API web permet à des outils externes d'accéder à votre installation
